Introduction to HTML by ConorC

DocType

Stands for 'Document Type Declaration' it defines the type of document for the browser as in which form of html for example <! doctype html>

HTML

Stands for 'HyperText Markup Language' it is the standard markup language for pages on browsers. The browser identifies opening <tags> and closing </tags>. <styles> identifies the style section in the head and contains CSS for the pages styling. </styles>

Head

The head is the part of the document that does not appear on the webpage as printed text but can contain the pages stylings through css or related links to web fonts and icons. Code for the head would look like the following:
<head> entries </head>
The head contains the document title <title>, CSS styling <style > and Metadata <meta>

Body

Is the main part of the html file and it contains all the information which will appear on the webpage like headings, text, images, links, tables, captions etc. The body is below and outside the head section. The code looks like

<body> typically lots of other code is contained within the body </body>

Title

Is the title of the document for the web browser to display for the document, located within the head section. On viewing this page in your browser, you will see the designated title appear in the tab on your browser. The title appears within the titles opening and closing tags and the code looks like this:   <title> Introduction to HTML by ConorC </title>

P

<p>this is a para</p>  

This is a paragraph that is typically used for regular font typed on the page as seen when the <p> tag is used with its closing tag </p>

Span

Span is an generic inline container for inline elements and content. It can be used to style text <span> text </span> in multiple ways, for example: in the following line a span will be used on the word Fox. The rabbit ran into the Fox

Div

<div> is a generic means of dividing up your web page into different sections. You could use a div for a header, a footer or sections within your page</div> for example <div id="divexample"> my wrapper text </div>

Formating elements are ways for you to style or format text in line with common features available in word.

Quote & Blockquote

Quote & Blockquote are similar features around inserting quotes. For quotes the tag q is used and the code would look like this
<q> Life is like a box of Chocolates</q>
Forrest Gumps said Life is like a box of chocolates

Blockquote displays the quote in a block seperate to the regular text as happens in some academic works. The code looks like this
<blockquote> Life is like a box of chocolates </blockquote>

Life is like a box of chocolates, you never know what you're gonna get.

Meta

Meta is data about data. Located in the head Meta can define the character set, page data, keywords in the content or the author. A common meta code identifies keywords and meta coding can look like this
<meta charset="utf-8">
<meta name="keywords" content="HTML, Head, Quote">

Special symbols and character entities:

Special symbols and character are encoded with unique identifiers. This can be useful as browsers can be thrown off by using symbols that are commonly used in the coding language and other times you might struggle to find a symbol on your keyboard. For these symbols below the coding will be displayed in bracket.

  • Copyright - displayed via (&copy;) ©
  • Registered Trademark - displayed via (&reg;) ®
  • Ampersand - displayed via (&amp;) &
  • Euro - displayed via (&euro;) €
  • Non-breaking space - displayed via (&nbsp;)  
  • less than - displayed via(&lt;) <
  • greater than - displayed via (&gt;) >
